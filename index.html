Projects

Multimodal Calibration Dataset - CALIBDB

A versatile multimodal calibration platform that uses robotic arms and a high-precision MoCap system to collect diverse extrinsic data between cameras and LiDAR sensors across both static and dynamic configurations.

Submitted to RSS 2025 (pending approval)

Plate Cleaning Robot

Developed an automated plate-cleaning system with an ABB IRB 120 robotic arm using inverse kinematics and PD control for spiral motions.

It features efficient trajectory planning and enhanced reliability through random plate positioning and robust task space design.

Autonomous Indoor Delivery Robot

Developed an autonomous indoor delivery robot using ROS for precise path planning. Integerated sensor fusion (Lidar, Wheel odom, IMU) with EKF for high accuracy localization.

Designed a PyQT-based GUI for easy user interation and voice control.

4WD Outdoor Navigation Robot

Developed a 4-wheel drive robot with 3D camera for real-time appearance-based mapping and navigation in Gps denied environments.

Autonomous Navigation of a Mecanum Robot Using DQN in ROS 2

This project implements DQN in ROS2 for autonomous navigation of a mecanum-wheeled robot. The robot learns to reach its target while avoiding obstacles in a simulated environment.
