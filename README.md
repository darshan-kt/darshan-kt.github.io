## 🛠️ Built With
This project is built from scratch using pure frontend technologies, designed to be lightweight, fast, and responsive:
- **HTML5:** Semantic structure and layout.
- **CSS3 (Vanilla):** Custom styling, grid/flexbox layouts, responsive design, and CSS animations.
- **JavaScript (Vanilla):** DOM manipulation, background slideshows, scroll reveals, and interactive elements.
- **FontAwesome & Google Fonts:** For vector icons and beautiful typography (Inter & Poppins).

## ⚙️ How to Run Locally

Since this is a static website with no backend framework required, running it locally is incredibly simple.

### Option 1: Direct File Open (Easiest)
Simply double-click the `index.html` file in your file explorer, and it will open in your default web browser.

### Option 2: Using Python Local Server
If you want to view it via a local development server (recommended to avoid CORS issues with some local assets), run this in your terminal:

```bash
# Navigate to the project folder
cd path/to/darshan-kt.github.io-main

# Start a local Python HTTP server
python3 -m http.server 8000
```
Then, open your browser and navigate to `http://localhost:8000`.

### Option 3: Using Node.js / NPX
If you have Node.js installed, you can use the `serve` package:
```bash
# Navigate to the project folder
cd path/to/darshan-kt.github.io-main

# Run the live server
npx serve .
```
Then, open the provided local URL (usually `http://localhost:3000`).
